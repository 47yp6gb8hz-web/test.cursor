AI-Powered Python 工具集
本仓库记录了我利用 Cursor AI 协同开发的多个 Python 自动化工具。作为统计学专业学生，我致力于通过技术手段解决实际的数据抓取与处理难题。

项目一、豆瓣 Top250 自动化抓取

功能：自动抓取豆瓣电影 Top250 的标题、评分及评价。
技术亮点：反爬机制攻克：针对 403 Forbidden 拦截，实现了 User-Agent 伪装及 Session 会话保持。
数据清洗：自动识别 Gzip 压缩 内容并修复 Encoding 编码 导致的中文乱码。
成果：生成结构化的 douban_top250.csv 文件。

项目二、通用网页标题提取器

功能：快速获取任意网页的元数据标题。
价值：作为自动化数据清洗的第一步，实现了对不同网页编码的鲁棒性支持。
